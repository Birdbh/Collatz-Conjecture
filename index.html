Collatz Conjecture
When it comes to the Collatz Conjecture it has been said by the prolific 20th century mathmatician Paul Erdos

"Mathimatics is not yet ready for such questions"

Yet the conjecture itself is really quite simple. Starting with any number, if it is even divide by 2. If it is odd Multiply by 3 and add 1. These seemingly simple rules result in a chaotic and random pattern that so far has been checked, always ends in 1. But no one has been able to prove it.
Take the Value of 10 for example. Starting at 10 we divide by 2 and get 5. This is odd so we multiply by 3 and add 1. This is 16 so we divide by 2 again. And get 8, then, 4, then 2, then 1.
